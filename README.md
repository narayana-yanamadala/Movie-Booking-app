## 📖 Project Overview

CineRush is a modern Movie Ticket Booking System built using **React.js**. The application allows users to browse movies, view details, select seats, verify bookings through OTP, upload payment proof, and receive booking confirmations through a clean and responsive interface.

---

## ✨ Features

✅ User Authentication

✅ Browse Movies

✅ Movie Details Page

✅ Seat Selection

✅ OTP Verification

✅ Payment Proof Upload

✅ Booking Confirmation

✅ Order Tracking

✅ Responsive Design

✅ Smooth Navigation

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/narayana-yanamadala/Movie-Booking-app.git
```

### Navigate to Project

```bash
cd Movie-Booking-app
```

### Install Dependencies

```bash
npm install
```

### Run Application

```bash
npm start
```
